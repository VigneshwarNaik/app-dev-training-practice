# Case Study 3: Hotel Room Booking System

## 📌 Overview

This case study explores how **Agile practices** are applied to develop a modern **Hotel Room Booking System**.

The system allows users to:

* Search hotels
* View room availability
* Book rooms
* Make online payments
* Receive booking confirmations

Agile helps teams build features incrementally, gather customer feedback, and adapt to changing requirements.

---

## 🎯 Learning Objectives

By the end of this case study, you will understand:

* Iterative development
* Continuous customer feedback
* Prioritization and planning
* Agile team collaboration
* Handling dynamic requirements

---

# 1. Problem Statement

A hotel company wants to build an online booking platform with features such as:

* User Registration
* Search Hotels
* View Available Rooms
* Book Rooms
* Online Payment
* Booking Confirmation

Customer requirements may change over time, making Agile a suitable approach.

---

# 2. Agile Approach

Agile develops software in **small iterations** called **Sprints**.

Each sprint delivers working software and collects customer feedback.

> **Build → Test → Review → Improve**

---

# 3. Iterative Development

# 4. Continuous Customer Feedback

Customers review the product after every sprint.

### Example Feedback

Customer says:

> "Please add filters for price and ratings."

The team adds these features in future sprints.

Benefits:

✅ Better product quality
✅ Higher customer satisfaction
✅ Faster improvements

---

# 6. Agile Team Collaboration

Agile teams work together closely.

A cross-functional team includes:

* Developers
* Testers
* UI/UX Designers
* Business Analysts
* Product Owner

Daily communication improves project success.

---

# 7. Challenges Addressed

## Dynamic Requirements

Customer needs change frequently.

Examples:

* Add discount coupons
* Add cancellation options
* Support multiple languages

Agile makes it easy to handle these changes.

---

## Integration with External Systems

The hotel booking system integrates with:

* Payment gateways
* Hotel databases
* Notification services
* Third-party APIs

Integration ensures real-time room availability.

---

## User Experience Optimization

Improve:

* Fast hotel search
* Mobile-friendly design
* Easy booking process

A better user experience increases customer satisfaction.

# 8. Benefits of Agile in Hotel Booking Systems

✅ Faster delivery

✅ Better software quality

✅ Reduced project risk

# 🔑 Key Takeaways

* Agile develops software in **Sprints**.
* Features are delivered incrementally.
* Customer feedback drives improvements.
* Teams prioritize important features first.
* Agile handles changing requirements effectively.
