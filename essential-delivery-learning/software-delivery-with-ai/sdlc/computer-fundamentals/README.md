# 🖥️ Computer Fundamentals

## 📌 Overview

This repository contains my learning notes on Computer Fundamentals. It covers the basic concepts of computers, their components, hardware, software, and the uses and benefits of software in everyday life.

---

## 📚 Topics Covered

### 1️⃣ What is a Computer?

A computer is an electronic device that accepts data as input, processes it according to instructions, stores it, and produces output.

### Basic Functions of a Computer

* Input
* Processing
* Storage
* Output

---

## 2️⃣ Components of a Computer

### ⌨️ Input Unit

Used to enter data into the computer.

**Examples:**

* Keyboard
* Mouse
* Scanner
* Microphone

### 🧠 Central Processing Unit (CPU)

The brain of the computer that performs calculations and controls operations.

**Components of CPU:**

* Arithmetic Logic Unit (ALU)
* Control Unit (CU)
* Memory Unit

### 💾 Memory Unit

Stores data and instructions.

#### RAM (Random Access Memory)

* Temporary memory
* Data is lost when power is turned off

#### ROM (Read Only Memory)

* Permanent memory
* Stores startup instructions

### 🗄️ Storage Unit

Stores data permanently.

**Examples:**

* Hard Disk Drive (HDD)
* Solid State Drive (SSD)
* Pen Drive

### 🖥️ Output Unit

Displays processed information.

**Examples:**

* Monitor
* Printer
* Speakers

---

## 3️⃣ Hardware and Its Types

Hardware refers to the physical parts of a computer that can be seen and touched.

### Types of Hardware

#### Input Hardware

* Keyboard
* Mouse
* Scanner

#### Output Hardware

* Monitor
* Printer
* Speakers

#### Processing Hardware

* CPU
* GPU

#### Storage Hardware

* HDD
* SSD
* Pen Drive

#### Communication Hardware

* Modem
* Network Card
* Wi-Fi Adapter

---

## 4️⃣ Software and Its Examples

Software is a collection of programs and instructions that tell a computer what to do.

### Types of Software

#### System Software

Manages computer hardware and system resources.

**Examples:**

* Windows
* Linux
* Device Drivers

#### Application Software

Helps users perform specific tasks.

**Examples:**

* Microsoft Word
* Google Chrome
* WhatsApp

#### Programming Software

Used to develop software applications.

**Examples:**

* Compiler
* Interpreter
* Visual Studio Code
* Eclipse

---

## 5️⃣ Uses and Benefits of Software

### Uses of Software

* Document creation
* Internet browsing
* Communication
* Online banking
* Entertainment

### Benefits of Software

#### ⚡ Automation

Automates repetitive tasks.

#### 🚀 Faster Processing

Performs tasks quickly and accurately.

#### 🌐 Better Communication

Enables communication through email, messaging, and video calls.

#### 💾 Data Storage

Stores large amounts of information.

#### 📈 Increased Productivity

Helps users complete tasks efficiently.
