# 📱 Application Lifecycle Management (ALM)

## 📌 Overview

This repository contains my learning notes and practical understanding of Application Lifecycle Management (ALM), Application Development Life Cycle (ADLC), and Software Development Life Cycle (SDLC). It covers the complete journey of an application from planning and development to deployment and maintenance.

---

## 🎯 What is an Application?

An application is software designed to perform specific tasks for users.

### Examples

* WhatsApp – Messaging
* Google Chrome – Web Browsing
* Amazon – Online Shopping
* Food Delivery Apps – Ordering Food

---

## 🔄 What is Application Lifecycle Management (ALM)?

Application Lifecycle Management (ALM) is the process of managing an application throughout its entire lifecycle.

### ALM Stages

1. Planning
2. Development
3. Testing
4. Deployment
5. Maintenance
6. Retirement

**Real-Life Example:**
A food delivery application starts as an idea, is developed and tested, deployed to users, maintained over time, and eventually replaced by a newer version.

---

## 🧩 Components of an Application

### 🎨 User Interface (UI)

The part of the application users interact with.

**Examples:**

* Login Page
* Home Screen
* Cart Page

### ⚙️ Business Logic

Contains rules and functionality.

**Examples:**

* Login validation
* Bill calculation
* Discount processing

### 🗄️ Database

Stores application data.

**Examples:**

* User information
* Orders
* Product details

### 🔗 Backend Services

Connects the UI with the database.

**Flow:**
User → UI → Backend → Database

---

## 🔨 Application Development Life Cycle (ADLC)

ADLC defines the steps involved in developing an application.

### Phases of ADLC

#### 1. Requirement Gathering

Collect user and business requirements.

#### 2. Design

Create system and UI designs.

#### 3. Development

Develop the application using programming languages.

#### 4. Testing

Identify and fix defects.

#### 5. Deployment

Release the application to users.

#### 6. Maintenance

Provide updates and support.

---

## 🚀 Application Kick-Off

The kick-off phase marks the beginning of a project.

### Activities

* Requirement discussion
* Scope definition
* Team allocation
* Project planning

---

## 🏁 Application End Stage

An application reaches its end stage when:

* Technology becomes outdated
* Business requirements change
* A new application replaces it

---

## 💻 Software Development Life Cycle (SDLC)

SDLC is a structured process used to build software.

### SDLC Phases

1. Analyze
2. Plan
3. Build
4. Test
5. Deploy
6. Maintain

### SDLC Workflow

```text
Analyze → Plan → Build → Test → Deploy → Maintain
```

---

## 📊 ALM vs ADLC vs SDLC

| Term | Full Form                          | Purpose                          |
| ---- | ---------------------------------- | -------------------------------- |
| ALM  | Application Lifecycle Management   | Complete application journey     |
| ADLC | Application Development Life Cycle | Application development process  |
| SDLC | Software Development Life Cycle    | Software development methodology |
