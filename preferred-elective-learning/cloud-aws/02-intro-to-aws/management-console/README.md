# AWS Management Console

## Introduction

The AWS Management Console is a web-based interface provided by Amazon Web Services (AWS) to manage cloud services using a browser.

It allows users to create, configure, monitor, and manage AWS resources easily without using command-line commands.

---

# What is AWS Management Console?

“AWS Management Console is a web-based graphical interface provided by Amazon Web Services (AWS) that allows users to manage AWS cloud services through a browser.

Using the console, we can create and manage services like EC2 virtual machines, S3 storage, databases, IAM users, and monitor resources without using command-line commands.

It is beginner-friendly and helps users easily access all AWS services from one dashboard.”

For example, if I want to create a virtual server, I can log in to the AWS Console, open EC2 service, click Launch Instance, and create the server using graphical options.”

# AWS Console vs AWS CLI

| AWS Management Console | AWS CLI                |
| ---------------------- | ---------------------- |
| Graphical Interface    | Command Line Interface |
| Easy for beginners     | Better for automation  |
| Uses mouse clicks      | Uses commands          |
| Good for learning      | Good for scripting     |

https://console.aws.amazon.com/

* Root user
* IAM user

---

# Example: Creating an EC2 Instance

## Step 1

Login to AWS Console

## Step 2

Search for "EC2"

## Step 3

Click "Launch Instance"

## Step 4

Choose:

* Operating System
* Instance Type
* Storage

## Step 5

Click "Launch"

AWS creates the virtual server.
