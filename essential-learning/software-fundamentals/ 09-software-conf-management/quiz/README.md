# Question 1

## Question
John bought a new Laptop with a high-end configuration. To protect his laptop and applications, he installed antivirus software.

This is an example of ______ Maintenance.

Options:
- Adaptive
- Preventive
- Perfective
- Corrective

---

## Correct Answer
✅ **Preventive Maintenance**

---

## Explanation
Preventive maintenance is done to avoid future problems before they occur.

Installing antivirus software helps protect the system from future threats like viruses and malware.

# Question 2

## Match the Correct Options

| Question | Correct Answer |
|---|---|
| Process that ensures that changes made are recorded and controlled | Change Management |
| Who authenticates that the change proposed is valid | Change Control Board |
| Process that ensures different versions of the project is managed | Configuration Management |
| The standard document where the requester fills the change in the change management process | Change Request Form |

---

## Explanation

### Change Management
Controls and records all changes made to software.

### Change Control Board (CCB)
Committee responsible for approving or rejecting changes.

### Configuration Management
Manages different software versions and configurations.

### Change Request Form
Document used to request software modifications.

# Question 3

## Question
Identify the features that are part of Software Configuration Management.

Options:
- a. Concurrency control
- b. Support management
- c. Synchronisation control
- d. Version management

---

## Correct Answers
✅ Concurrency control  
✅ Synchronisation control  
✅ Version management

---

## Explanation

### Concurrency Control
Allows multiple developers to work safely.

### Synchronisation Control
Prevents overwriting of parallel changes.

### Version Management
Maintains multiple software versions.

❌ Support management is not an SCM feature.

# Question 5

## Question
The entry door of the Server room inside the company can be considered equivalent to ______ concept in configuration management.

Options:
- artifact
- version
- baseline
- SRS

---

## Correct Answer
✅ Baseline

---

## Explanation

A baseline is a formally approved version that serves as the foundation for future development.

Just like a server room entry is controlled and protected, a baseline acts as a controlled reference point in SCM.
