# 🚀 What is Application Maintenance?

Application Maintenance refers to the activities performed **after software has been deployed** to users.

The goal is to ensure that the application continues to work efficiently, securely, and meets changing user needs.

### Simple Definition

> Application Maintenance is the process of modifying and updating software after deployment.

# 🛠️ Types of Application Maintenance

There are four major types of maintenance:

1. Corrective Maintenance
2. Adaptive Maintenance
3. Preventive Maintenance
4. Perfective Maintenance

---

# 1️⃣ Corrective Maintenance

## What is it?

Corrective Maintenance focuses on **fixing bugs and defects** found after deployment.

### Purpose

* Correct errors
* Resolve failures
* Improve stability

### Examples

* Fix login failure
* Correct calculation errors
* Repair broken links

### Real-Life Example

Users report that the payment page crashes.

Developers fix the bug and release an update.

---

# 2️⃣ Adaptive Maintenance

## What is it?

Adaptive Maintenance modifies software to **adapt to changes in the environment**.

### Purpose

* Support new technologies
* Comply with new regulations
* Work with updated operating systems

### Examples

* Updating an app for a new Android version
* Integrating a new payment gateway
* Adapting to government regulations

### Real-Life Example

A banking app updates its system to comply with new security laws.

---

# 3️⃣ Preventive Maintenance

## What is it?

Preventive Maintenance improves software to **prevent future problems**.

### Purpose

* Reduce future risks
* Improve code quality
* Enhance maintainability

### Examples

* Refactoring code
* Updating libraries
* Removing unused code

### Real-Life Example

Developers optimize database queries before performance issues occur.

---

# 4️⃣ Perfective Maintenance

## What is it?

Perfective Maintenance enhances software by **adding new features or improving existing ones**.

### Purpose

* Improve user experience
* Add functionality
* Increase efficiency

### Examples

* Dark mode feature 🌙
* Improved user interface
* Faster search functionality

### Real-Life Example

An e-commerce app adds product recommendations for users.

---

# 📊 Comparison of Maintenance Types

| Type       | Purpose               | Example             |
| ---------- | --------------------- | ------------------- |
| Corrective | Fix bugs              | Resolve login error |
| Adaptive   | Adjust to changes     | Support new OS      |
| Preventive | Prevent future issues | Refactor code       |
| Perfective | Improve features      | Add dark mode       |
