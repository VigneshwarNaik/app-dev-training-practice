# 📌 Question 1 – Flowchart Error

## Question
Is the given flowchart correct?

## Given Flowchart

```text
 ┌─────────┐
 │  Start  │
 └────┬────┘
      │
 ┌─────────┐
 │ READ a  │
 └────┬────┘
      │
 ┌─────────┐
 │ sum=a+b │
 └────┬────┘
      │
 ┌─────────┐
 │ PRINT   │
 └────┬────┘
      │
 ┌─────────┐
 │  Stop   │
 └─────────┘
```

## Options

```text
a. The symbol for process is incorrect
b. The symbol for reading input from the user is incorrect
c. The flowchart has no error
d. The symbol for start/stop is incorrect
```

## ✅ Correct Answer
**b. The symbol for reading input from the user is incorrect**

## 💡 Explanation
- Input/output operations use **Parallelogram** symbol.
- READ operation should use parallelogram.

```text
a.
```

# 📌 Question 4 – Valid Variable Names

## Question
Identify the meaningful variable names which can be used.

## Options

```text
a. user1
b. registerNumber
c. 1num
d. user name
```

## ✅ Correct Answers
- **a. user1**
- **b. registerNumber**

### Variable Naming Rules
- Cannot start with number
- Cannot contain spaces
- Can contain letters and numbers

- `1num`
