

# Question 1

## Question

Choose the correct options to complete the pseudo-code and determine whether the number is positive, zero, or negative.

```text
BEGIN
DECLARE number
READ number

IF _________
    PRINT "Number is positive"

IF _________
    PRINT "Number is zero"

IF _________
    PRINT "Number is Negative"

END IF
END
```

## Options

a. `number>0, number==0, number==0`

b. `number>0, number==0, number<0`

c. `number>0, number>0, number==0`

d. `number<0, number>0, number==0`

## ✅ Correct Answer

**b.** `number>0, number==0, number<0`

## 💡 Explanation

* Positive numbers are greater than 0.
* Zero is equal to 0.
* Negative numbers are less than 0.
