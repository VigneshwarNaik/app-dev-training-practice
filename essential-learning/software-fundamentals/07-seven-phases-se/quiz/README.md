# Check Your Understanding – Detailed Quiz Questions & Answers

# Question 1

## Question
Which of the following options are the steps involved in Requirements Analysis?

---

## Options

### a. Analysis of the gathered requirements

### b. Requirements Design

### c. Requirements Specification

### d. Requirements Elicitation

---

# ✅ Correct Answers

- ✅ a. Analysis of the gathered requirements
- ✅ c. Requirements Specification
- ✅ d. Requirements Elicitation

---

# ❌ Incorrect Answer

- ❌ b. Requirements Design

---

# Detailed Explanation

## What is Requirements Analysis?

Requirements Analysis is the process of:
- Collecting customer requirements
- Studying requirements carefully
- Removing ambiguities
- Documenting requirements properly

---

# Steps Involved in Requirements Analysis

## 1. Requirements Elicitation

### Meaning
Requirements are collected from:
- Customers
- Stakeholders
- Users

### Techniques Used
- Interviews
- Surveys
- Observation
- Workshops

---

## 2. Analysis of Gathered Requirements

### Meaning
Collected requirements are:
- Studied carefully
- Validated
- Checked for conflicts
- Organized properly

---

## 3. Requirements Specification

### Meaning
Final requirements are documented in:
```text
SRS (Software Requirement Specification)
```

---

# Why is “Requirements Design” Wrong?

Requirements Design belongs to:
```text
Software Design Phase
```

NOT:
```text
Requirements Analysis Phase
```

---

# Requirement Analysis Flow

```text
Requirements Elicitation
        ↓
Requirements Analysis
        ↓
Requirements Specification
```

---

# Question 2

## Question
From the below options, identify the role of the system analyst.

---

## Options

### a. Creates high level test cases

### b. Writes pseudo code for the given module

### c. Creates HLD document

### d. Creates SRS

---

# ✅ Correct Answer

- ✅ d. Creates SRS

---

# Detailed Explanation

## Who is a System Analyst?

A System Analyst is responsible for:
- Understanding business requirements
- Communicating with clients
- Preparing requirement documents
- Creating SRS

---

# Main Responsibilities of System Analyst

| Responsibility | Description |
|---|---|
| Requirement Gathering | Collect customer needs |
| Requirement Analysis | Analyze requirements |
| Requirement Documentation | Prepare SRS |
| Client Communication | Clarify requirements |

---

# What is SRS?

SRS stands for:
```text
Software Requirement Specification
```

It contains:
- Functional Requirements
- Non-Functional Requirements
- Constraints
- System Expectations

---

# Why Other Options are Wrong?

| Option | Reason |
|---|---|
| Creates high level test cases | Testing team responsibility |
| Writes pseudo code | Developer responsibility |
| Creates HLD document | Architect/Designer responsibility |

# Question 3

## Question
Identify the correct statements from the below options.

---

## Options

### a. High level design, Low level design followed by Analysis is performed

### b. Analysis or High level design or Low level design can be performed in any order

### c. Analysis is performed followed by High level design and then Low level design

### d. Analysis is performed followed by low level design and then high level design

---

# ✅ Correct Answer

- ✅ c. Analysis is performed followed by High level design and then Low level design

---

# Detailed Explanation

## Correct Software Engineering Flow

```text
Requirement Analysis
        ↓
High Level Design (HLD)
        ↓
Low Level Design (LLD)
```

---

# Step-by-Step Explanation

## 1. Requirement Analysis

### Purpose
Understand:
- Customer needs
- System requirements
- Business problems

---

## 2. High Level Design (HLD)

### Purpose
Create:
- System architecture
- Module design
- Database design
- Module communication

---

## 3. Low Level Design (LLD)

### Purpose
Create:
- Algorithms
- Pseudocode
- Detailed logic
- Function implementation

---

# Why Other Options are Wrong?

| Option | Reason |
|---|---|
| HLD → LLD → Analysis | Wrong sequence |
| Any order | SDLC follows proper sequence |
| Analysis → LLD → HLD | HLD always comes before LLD |
