# Introduction to Generative Models

Generative Models are the foundation of Generative AI.

They learn patterns from existing data and create new data that looks similar to the original data.

### Simple Example

If a Generative Model learns from thousands of cat images, it can create a new cat image that never existed before.

---

# What are Generative Models?

Generative Models are AI models that can generate new content such as:

- Text
- Images
- Audio
- Videos
- Code

They learn from existing data and create new data with similar patterns.

### Example

After learning many stories, a Generative Model can write a new story.

After learning many images, it can create a new image.

---

# Generative vs Discriminative Models

There are two common types of AI models:

## Generative Models

Generative Models create new data.

### Examples

- ChatGPT generating text
- AI creating images
- Music generation

### Question They Answer

**"How can I create new data?"**

---

## Discriminative Models

Discriminative Models classify or identify data.

### Examples

- Spam email detection
- Face recognition
- Disease prediction

### Question They Answer

**"Which category does this data belong to?"**

---

## Simple Comparison

| Generative Model | Discriminative Model |
|------------------|---------------------|
| Creates new data | Classifies data |
| Learns patterns | Learns differences |
| Generates text and images | Identifies categories |
| Example: ChatGPT | Example: Spam Filter |

---

# Probability Distributions

Probability Distribution shows how likely different outcomes are.

Generative Models use probability to predict what comes next.

### Simple Example

If you type:

```
I am going to
```

The model predicts likely next words such as:

- school
- market
- office

It chooses words based on probability.

### Purpose

Probability helps AI generate realistic content.

---

# Data Generation Concepts

Data Generation means creating new data after learning patterns from existing data.

### Steps

#### Step 1: Learn Data

The model studies existing data.

### Example

- Thousands of images
- Millions of sentences
- Large amounts of audio

#### Step 2: Find Patterns

The model learns common patterns and relationships.

#### Step 3: Generate New Data

The model creates new content based on what it learned.

### Example

Learn:
- Many cat images

Generate:
- A new cat image that never existed before
