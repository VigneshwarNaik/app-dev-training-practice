# Question 1

## Question
Which of the following options are valid for the relationship between the configuration objects?

### Options
- a. A double-headed straight arrow indicates an interrelationship
- b. A double-headed straight arrow indicates compositional relation
- c. A curved arrow indicates an interrelationship
- d. A curved arrow indicates a compositional relation

## Correct Answers
✅ a. A double-headed straight arrow indicates an interrelationship  
✅ d. A curved arrow indicates a compositional relation

## Explanation
- A **double-headed straight arrow** shows an **interrelationship** between configuration objects.
- A **curved arrow** shows a **compositional relationship**.

# Question 2

## Question
Match the following facts about Version Management.

| Statement | Correct Match |
|---|---|
| Lock a file | Serialized changes to file |
| When a member of the team wants his code to work in isolation | Create branches |
| If a file is changed and we want to roll back to the previous version | Automatic backup |

## Explanation
- **Serialized changes to file** prevents multiple users from editing the same file at the same time.
- **Create branches** allows developers to work independently.
- **Automatic backup** helps restore previous versions.
