# ☁️ Introduction to AWS Cloud - PostQuiz


# 📘 Question 1

## Question
AWS Cloud's global footprint enables you to easily deploy applications to consumers all around the world.

### Options
- True
- False

## ✅ Correct Answer
**True**

## 💡 Explanation
AWS has data centers located in multiple regions across the world. This global infrastructure helps businesses deploy applications closer to users, improving performance and reducing latency.

# 📘 Question 2

## Question
How does cloud computing compare to traditional data centers in terms of obtaining new resources?

### Options
- Traditional data centers provide unlimited resources instantly.
- Cloud computing enables access to new resources in minutes.
- Cloud computing takes weeks to obtain new resources.
- Traditional data centers are more agile in resource acquisition.

## ✅ Correct Answer
**Cloud computing enables access to new resources in minutes.**

## 💡 Explanation
Cloud computing provides on-demand resources. Users can create servers, storage, and services quickly without waiting for hardware installation.

# 📘 Question 5

## Question
Which of the following consists of one or more data centres?

### Options
- AvailabilityZone
- Location
- Region
- Origin

## ✅ Correct Answer
**AvailabilityZone**

## 💡 Explanation

# 📘 Question 8

## Question
_______ eliminates the management of the underlying infrastructure (usually hardware and operating systems) and allows customer to focus on deploying and managing the applications.

### Options
- SaaS
- XaaS
- PaaS
- IaaS

## ✅ Correct Answer
**PaaS**

## 💡 Explanation
Platform as a Service (PaaS) manages hardware and operating systems, allowing developers to focus only on application development.

# 📘 Question 9

## Question
When could a company decide to use a hybrid deployment model for cloud computing?

### Options
- To use only cloud-based data and analytics services
- To fully comply with government regulations
- To eliminate all legacy applications
- To maintain legacy applications on premises

## ✅ Correct Answer
**To maintain legacy applications on premises**

## 💡 Explanation
Hybrid cloud combines on-premise infrastructure with cloud services. Companies often use it to continue using legacy systems while adopting cloud technology.
