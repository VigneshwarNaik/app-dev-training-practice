# ☁️ What is AWS?

## Amazon Web Services (AWS)

AWS is a cloud computing platform provided by Amazon.

It offers:
- Computing power
- Storage services
- Networking
- Databases
- Security
- Analytics
- Artificial Intelligence services

AWS allows organizations to:
- Host applications
- Store data
- Scale resources
- Reduce infrastructure costs

---

# 🌐 Cloud Computing

Cloud Computing means delivering computing services over the internet.

## Examples of Cloud Services
- Servers
- Databases
- Storage
- Networking
- Software

Instead of buying physical hardware, users can access resources online.

---

# ☁️ Types of Cloud Models

## 1. On-Premises
Infrastructure is maintained inside the organization.

### Features
- Full control
- Higher maintenance cost
- Requires physical hardware

---

## 2. Hybrid Cloud
Combination of:
- On-premises infrastructure
- Cloud services

### Features
- Flexible
- Secure
- Better scalability

---

## 3. All-in Cloud
Everything is hosted completely on the cloud.

### Features
- No physical infrastructure
- Highly scalable
- Cost-effective

---

# 🌍 AWS Global Infrastructure

AWS infrastructure includes:

- Regions
- Availability Zones
- Edge Locations

---

## AWS Region
A geographical area containing multiple data centers.

### Example
- Mumbai Region
- Singapore Region
- US-East Region

---

## Availability Zone (AZ)
An isolated data center within a region.

### Purpose
- High availability
- Fault tolerance
- Disaster recovery

---

## Edge Locations
Used for:
- Content delivery
- Faster access
- Low latency

---

# ✅ Benefits of AWS Cloud

## 1. Cost Savings
Pay only for resources used.

## 2. Scalability
Increase or decrease resources as needed.

## 3. High Availability
Services remain available even during failures.

## 4. Security
Strong cloud security mechanisms.

## 5. Flexibility
Supports various business needs.

## 6. Global Reach
Services available worldwide.

# ⚙️ AWS Core Services

AWS provides many cloud services.

---

# 💻 Compute Services

## Amazon EC2
Elastic Compute Cloud provides virtual servers.

### Uses
- Host websites
- Run applications
- Deploy software

---

# 💾 Storage Services

## Amazon S3
Simple Storage Service used for storing files and data.

### Features
- Secure storage
- High durability
- Backup support

---

# 🗄️ Database Services

## Amazon RDS
Relational Database Service for managing databases.

### Supports
- MySQL
- PostgreSQL
- Oracle
- SQL Server

---

# 🌐 Networking Services

## Amazon VPC
Virtual Private Cloud helps create isolated cloud networks.

### Features
- Secure communication
- Network control
- Resource isolation

---

# 🔐 Security and Compliance

## Features
- Identity and Access Management (IAM)

- Compliance certifications

# 💰 Billing and Pricing

AWS follows:
```text
Pay-As-You-Go
```

Users pay only for the services they consume.

---

# 💵 Pricing Models

## On-Demand
Pay for usage without long-term commitment.

## Reserved Instances
Lower cost with long-term reservation.

## Spot Instances
Unused AWS capacity available at lower prices.
