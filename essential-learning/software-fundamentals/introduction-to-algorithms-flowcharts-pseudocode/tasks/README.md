# ✅ Task 1 — Hello World

## 📌 Question

Write a pseudocode for a simple program that displays the message:

```text
Hello World
```

---

## ✅ Answer

```text
BEGIN

PRINT "Hello World"

END
```

---

## ✅ Explanation

| Statement | Purpose |
|---|---|
| BEGIN | Starts the program |
| PRINT | Displays output |
| END | Stops the program |

# ✅ Task 2 — Speed Computation

## 📌 Question

Jack crosses a street that is **X meters** long in **Y minutes**.

Write a pseudocode to calculate speed in **km/hr**.

### 📌 Formula

```text
Speed = Distance / Time
```

### 📌 Conversion

```text
1 m/min = 0.06 km/hr
```

---

## ✅ Answer

```text
BEGIN

READ X, Y

SET speed = (X / Y) * 0.06

PRINT speed

END
```

---

## ✅ Explanation

| Step | Explanation |
|---|---|
| READ X, Y | Takes distance and time |
| X / Y | Calculates speed in m/min |
| * 0.06 | Converts to km/hr |
| PRINT speed | Displays result |

---

## ✅ Example

```text
X = 500
Y = 10

speed = (500 / 10) * 0.06

speed = 3 km/hr
```

# ✅ Task 3 — Estimation of Total Cost

## 📌 Question

Bentley Motors Limited wants to estimate the total cost of a car by adding:

- 15% Excise Duty
- 10% Sales Tax
- 1% Road Tax

Write a pseudocode to calculate the final total cost.

---

## ✅ Answer

```text
BEGIN

READ basicCost

SET exciseDuty = basicCost * 0.15

SET salesTax = basicCost * 0.10

SET roadTax = basicCost * 0.01

SET totalCost = basicCost + exciseDuty + salesTax + roadTax

PRINT totalCost

END
```

---

## ✅ Explanation

| Step | Explanation |
|---|---|
| READ basicCost | Takes basic cost |
| * 0.15 | Calculates excise duty |
| * 0.10 | Calculates sales tax |
| * 0.01 | Calculates road tax |
| totalCost | Adds all costs |
| PRINT totalCost | Displays final amount |

---

## ✅ Example

```text
basicCost = 100000

exciseDuty = 15000
salesTax = 10000
roadTax = 1000

totalCost = 126000
```
