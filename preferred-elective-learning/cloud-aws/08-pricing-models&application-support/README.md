# AWS Pricing Models and Application Support

## Introduction

AWS provides different pricing models to help customers use cloud services based on their needs and budget.

AWS also provides application support services to help customers solve technical issues and manage AWS resources efficiently.

Understanding AWS pricing models helps organizations reduce costs, while AWS support plans help resolve problems quickly.

---

# What is AWS Pricing?

AWS follows a:

```text id="d1r4sj"
Pay-As-You-Go Model
```

This means:

```text id="d8v0hf"
Pay only for what you use
```

Just like paying an electricity bill, you pay only for the resources you consume.

---

# Why AWS Pricing Models?

Different customers have different requirements.

Example:

* Some need servers for a few hours.
* Some need servers for years.
* Some need flexible usage.
* Some need lower costs.

AWS provides different pricing options for these situations.

---

# AWS Pricing Models

## 1. On-Demand Pricing

### What is On-Demand?

Pay for resources only when they are running.

### Simple Meaning

```text id="2m7z2n"
Use Now → Pay Now
```

### Example

You launch an EC2 instance for:

```text id="jlwm5e"
5 Hours
```

You pay only for those 5 hours.

### Advantages

* No long-term commitment
* Easy to use
* Flexible

### Best For

* Learning AWS
* Testing
* Small Projects

---

# 2. Reserved Instances (RI)

### What are Reserved Instances?

You reserve AWS resources for:

```text id="hiv3pi"
1 Year
or
3 Years
```

and receive a discount.

### Simple Meaning

```text id="j2h9rz"
Long-Term Use = Lower Cost
```

### Example

A company knows it will use a server for 3 years.

Instead of paying every day:

```text id="k5m4bn"
Reserve It
```

and save money.

### Advantages

* Lower Cost
* Predictable Billing

### Best For

* Long-term projects
* Production applications

---

# 3. Savings Plans

### What are Savings Plans?

Savings Plans provide discounts when you commit to a certain amount of usage.

### Simple Meaning

```text id="qq8eeh"
Commit Usage → Get Discount
```

### Advantages

* Flexible
* Lower Cost
* Works across multiple services

### Best For

* Regular AWS users

---

# 4. Spot Instances

### What are Spot Instances?

Spot Instances use AWS's unused capacity at a lower price.

### Simple Meaning

```text id="c4ok2s"
Unused AWS Servers
=
Very Cheap
```

### Important

AWS can stop the instance at any time.

### Example

Use Spot Instances for:

* Testing
* Batch Processing
* Data Analysis

### Advantages

* Up to 90% cheaper

### Disadvantages

* Can be interrupted anytime

---

# 5. Free Tier

### What is AWS Free Tier?

AWS provides free services for beginners.

### Examples

* EC2 t3.micro
* Amazon S3
* Lambda
* DynamoDB

### Benefits

* Learn AWS for free
* Practice cloud services

### Best For

* Students
* Beginners
* Training

---

# AWS Pricing Model Summary

| Pricing Model      | Description           | Best For               |
| ------------------ | --------------------- | ---------------------- |
| On-Demand          | Pay as you use        | Learning and Testing   |
| Reserved Instances | Long-term commitment  | Production             |
| Savings Plans      | Commit usage and save | Regular Users          |
| Spot Instances     | Unused AWS capacity   | Testing and Batch Jobs |
| Free Tier          | Free usage limits     | Beginners              |

# What is AWS Application Support?

AWS Application Support helps customers solve technical problems and get assistance from AWS experts.

Support is useful when:

* Applications fail
* Servers stop working
* Configuration issues occur
* Performance problems arise

---

# AWS Support Plans

AWS provides different support plans.

---

# 1. Basic Support

### What is Basic Support?

Free support provided to all AWS customers.

### Includes

* Documentation
* AWS Knowledge Center
* Service Health Dashboard
* Billing Support

### Cost

```text id="zx7szg"
Free
```

---

# 2. Developer Support

### What is Developer Support?

Support for developers testing and building applications.

### Includes

* Technical Support
* Email Support
* Best Practice Guidance

### Best For

* Developers
* Small Projects

---

# 3. Business Support

### What is Business Support?

Provides faster response times and technical guidance.

### Includes

* 24/7 Support
* Phone Support
* Trusted Advisor

### Best For

* Businesses
* Production Applications

---

# 4. Enterprise Support

### What is Enterprise Support?

Highest level of AWS support.

### Includes

* 24/7 Expert Support
* Dedicated Technical Account Manager (TAM)
* Architecture Guidance

### Best For

* Large Enterprises
* Critical Applications

# Conclusion

AWS offers multiple pricing models to suit different customer needs. On-Demand pricing provides flexibility, Reserved Instances and Savings Plans reduce costs, Spot Instances offer very low pricing, and the Free Tier helps beginners learn AWS. AWS also provides support plans ranging from Basic Support to Enterprise Support, helping customers resolve issues and manage applications effectively.
